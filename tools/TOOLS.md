## KB Vault + CLI = vaultli

- File-based knowledge base management with YAML frontmatter, sidecar markdown, and a derived `INDEX.jsonl`
- Best for making docs, queries, templates, runbooks, and skills agent-discoverable without introducing a database
- Start with `tools/vaultli/README.md` for the full guide and `tools/vaultli/SKILL.md` for the agent-first workflow



## Clipboard + CLI = clipli

- macOS clipboard intelligence for capture, templated paste, Excel-native HTML generation, and format conversion
- Best when an agent needs to inspect the current clipboard, preserve formatting, or preview rich output before writing it back
- Start with `tools/clipli/clipli/SKILL.md` for the safe agent workflow and `tools/clipli/CLIPLI_SPEC.md` for the full surface
